# ElGamal Cryptosystem

Python implementation of ElGamal Cryptosystem with GUI

## Getting Started

All you need to try this out is just clone repository to your PC and run `gui.py` script using your python interpreter.

### Requirements
  * Python 3.7

  Python libraries:
  * Tkinter
